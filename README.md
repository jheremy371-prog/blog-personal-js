# 📘 Blog Personal

Proyecto de Blog web con autenticación básica y operaciones CRUD completas utilizando JavaScript puro y LocalStorage para la persistencia de datos. 

## Características
- Registro e inicio de sesión de usuarios.
- Creación, lectura, edición y eliminación de publicaciones.
- Persistencia local sin necesidad de backend.
- Despliegue mediante GitHub Pages.
